# LinuxQuest

Interactive Linux terminal learning platform. Frontend + backend + Docker support.

## Project Structure

```
LinuxQuest/
├── frontend/          # React + Vite + Tailwind
├── backend/           # Node.js + Express
├── docker/            # Docker configurations
└── docs/              # Documentation
```

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## Technology Stack

**Frontend:**
- React 18
- Vite
- TailwindCSS
- React Router
- Zustand (state management)
- xterm (terminal emulation)

**Backend:**
- Node.js + Express
- PostgreSQL
- Socket.io (real-time)
- JWT Authentication
- bcrypt (password hashing)
- Express Rate Limit

## Development

Start both frontend and backend for full development environment.
